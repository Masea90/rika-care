# RIKA Care

AI-powered beauty guidance platform with personalized skincare and haircare recommendations.

## Features
- AI skin/hair analysis
- Personalized product recommendations  
- Daily routine tracking with streaks
- Community sharing
- Points & rewards system
- AI chatbot assistant

## Tech Stack
- Backend: Node.js + Express + SQLite
- Frontend: Vanilla HTML/CSS/JavaScript
- AI: Rule-based responses (LLM-ready)

## Local Development
```bash
cd backend
npm install
node server.js
```
Open `web/rika-care.html` in browser.

## Deployment
Ready for Heroku, Railway, or Render deployment.
